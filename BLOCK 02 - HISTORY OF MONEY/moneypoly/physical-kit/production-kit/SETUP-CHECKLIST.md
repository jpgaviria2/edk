# Setup Checklist

## Day before
- print all kit files
- sort printables by table number
- bag 8 table kits
- confirm staffing assignments
- test projector and timer
- preload Moneypoly pages on facilitator device

## 90 minutes before doors
- place table signs 1-8
- place banker zone materials at each table pair
- stage central supply station
- tape room signage
- place score clipboards

## 45 minutes before doors
- load intro screen
- test countdown timer visibility
- count starting goods by table
- brief staff using staffing docs

## 15 minutes before doors
- place role packets at tables
- place pens/name tags
- verify event decks and wallet cards are present
- hold final staff huddle
