# Event Deck Rules

## Use
- draw one event per table pair at each major transition or every 3 trade attempts
- banker applies the effect immediately
- event master keeps the room roughly synchronized

## Event categories
### Environmental
- Drought: water prices double
- Flood: shelter loss
- Pest outbreak: food loss

### Economic
- Shell rush: more shells enter, shell prices rise
- Counterfeit scare: gold verification needed
- Bank run: slips redeem only if reserves exist

### Political
- Freeze order: selected fiat balances cannot move this round
- Confiscation: exposed gold can be seized

### Bitcoin specific
- Money printer attempt: no change to sats supply
- Lost phone: hot-wallet holders lose spending access this round
- Seed phrase check: cold-storage players keep savings if phrase retained
