# AV / Projection Script

## Before doors open
- load root EDK page
- preload Block 02 page
- preload Moneypoly page
- confirm service worker/offline shell loads
- open countdown timer tab
- set screen brightness high

## Projection sequence

### Intro
Project Block 02 overview or Moneypoly home.
Say:
"You are about to test different kinds of money against the same human problem."

### Phase transitions
Use a simple title card or say clearly:
- Barter
- Commodity Money
- Gold
- Credit & Ledgers
- Fiat
- Bitcoin

### During play
Project only what supports the current phase:
- instructions modal
- comparison view
- teacher mode prompts

Avoid overloading the room with small text.

### Debrief
Project Compare Eras and score categories:
- survival
- purchasing power
- seizure/freeze risk
- portability/divisibility
- generational wealth

## Backup plan
If internet fails:
- use cached PWA pages already loaded
- if laptop dies, switch to printed facilitator cheat sheet and physical boards only

## Audio cues
Optional but useful:
- 1 bell = one minute left
- 2 bells = freeze and listen
- 3 bells = transition now
