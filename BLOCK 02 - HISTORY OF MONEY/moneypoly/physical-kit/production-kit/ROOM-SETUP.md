# Room Setup

## Layout

### Front of room
- projector / screen
- lead facilitator position
- AV table
- central countdown timer visible to all
- whiteboard / flipchart

### Floor
- 8 numbered tables
- enough spacing for bankers to move quickly
- one supply station at back or side wall
- one overflow materials bin

## Table layout
Each table should have:
- table number sign
- 5 team positions
- 1 banker zone
- 1 token tray area
- 1 event discard area
- 1 price board stand-up card
- 1 score sheet clipboard

## Supply station
Keep separate bins for:
- barter goods
- commodity money
- gold
- credit slips and ledgers
- fiat notes
- sats and wallet cards
- spare pens and tape

## Projection plan
Project these views in order:
1. host intro / rules
2. phase title + timer
3. Moneypoly teacher mode visuals during transitions
4. final compare eras screen during debrief

## Traffic rule
No participant should need to walk across the room to play. All gameplay stays table-local except when a facilitator deliberately creates a branch or exchange moment.
