# History of Money Website

A website-first educational experience for EDK Block 1: History of Money.

## Direction

This is no longer framed as only a Fedi mini app. It is a static educational website that can still be wrapped later, but the primary goal is now teaching clarity:

- short structured lessons
- practical examples
- interactive demonstrations
- facilitator support
- curriculum alignment with the My First Bitcoin Diploma

## What it includes

- Lesson path:
  - Why barter breaks
  - The three functions of money
  - What makes money good
  - From commodity money to fragile trust
  - Why Bitcoin belongs in the story
- Practical examples:
  - island barter chain / double coincidence of wants
  - function-of-money classifier
  - good-money scorecard
  - shell supply / purchasing power demo
  - coin debasement / trust demo
- Facilitator mode:
  - suggested 50-minute workshop flow
  - class prompts and section timing

## Curriculum alignment

Based on these My First Bitcoin Diploma sections:

- `1-what-is-money/3-functions-of-money.md`
- `1-what-is-money/4-properties-of-money.md`
- `2-the-history-of-money/0-introduction.md`
- `2-the-history-of-money/1-from-barter-to-modern-currency.md`

## Files

- `index.html` — static document shell
- `styles.css` — website-first responsive design
- `app.js` — lesson navigation and interactive examples
