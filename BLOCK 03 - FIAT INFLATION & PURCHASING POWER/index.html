Fiat, Inflation & Purchasing Power

Feel what happens when the number of money units grows faster than the goods people want to buy.

StartHost this lesson
Learner progress

One block. One core idea. One next action.

Lesson path

Short, guided, and hard to get lost in.

1

Start with scarcity

Three people want one bottle of water. Their bids reveal demand.

2

Add new money

Everyone receives more dollars, but the number of bottles does not change.

3

Notice purchasing power

People may hold more currency units while each unit buys less than before.

Try it

What usually happens?

Everyone has more dollars, but there is still only one bottle of water.

Reflect

Real-world check

Name one thing that costs more than it used to. Did income rise at the same speed?
