# Block 7 Physical Kit

Hands-on classroom and live-event materials for **Transactions: Send & Receive**.

## Bitcoin Transaction Demo

Path: `bitcoin-transaction-demo/`

Use this kit as a **center-stage mirrored network**: the moderator demonstrates one transaction scenario on the projected ledger while all 8 tables recreate the same transaction locally.

Each table shows the full lifecycle:

1. Wallet selects a UTXO and signs a transaction
2. Nodes validate signature, ownership, amount, and double-spend risk
3. Valid transactions enter the mempool
4. Mempool sorts by fee
5. Miners roll dice to find a block
6. Nodes validate the candidate block
7. The table updates the shared blockchain ledger and compares it to the center-stage ledger

Designed for **8 tables × 10 people = 80 participants**. Recommended table roles: **3 wallets, 3 nodes, 2 miners, 1 mempool manager, 1 table captain**.
