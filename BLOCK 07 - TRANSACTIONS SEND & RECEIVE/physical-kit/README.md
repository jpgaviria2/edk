# Block 7 Physical Kit

Hands-on classroom and live-event materials for **Transactions: Send & Receive**.

## Bitcoin Transaction Demo

Path: `bitcoin-transaction-demo/`

Use this kit to show the full lifecycle of a Bitcoin transaction:

1. Wallet creates and signs a transaction
2. Nodes validate it
3. Valid transactions enter the mempool
4. Mempool sorts by fee
5. Miners roll dice to find a block
6. Nodes validate the block
7. The table updates the shared blockchain ledger

Designed for **8 tables × 10 people = 80 participants**.
