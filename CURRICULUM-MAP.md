# EDK Curriculum Map

This repository now includes an attributed copy of the English My First Bitcoin Bitcoin Diploma curriculum under:

`curriculum/my-first-bitcoin/bitcoin-diploma/en/`

## Purpose

The imported curriculum is the educational source layer for building EDK practical games, activities, and interactive lessons. The EDK block folders remain the product/design layer.

## Block mapping

| EDK Block | Main curriculum sources | Practical build direction |
|---|---|---|
| Block 1 — History of Money | `1-what-is-money`, `2-the-history-of-money`, `3-what-is-fiat-money`, `4-how-problems-lead-to-solutions` | Barter, money functions/properties, inflation, debasement, Bitcoin bridge |
| Block 2 — Hashing, Nodes & Mining | `5-what-is-bitcoin`, `8-how-bitcoin-works`, `9-how-does-bitcoin-mining-work` | Hashing, blocks, nodes, miners, proof of work, mempool |
| Block 3 — Transaction Fees & Speed | `7-using-bitcoin-in-daily-life`, `9-how-does-bitcoin-mining-work` | Fee market, mempool, Lightning, everyday payments |
| Block 4 — Byzantine Generals | `4-how-problems-lead-to-solutions`, `5-what-is-bitcoin`, `8-how-bitcoin-works`, `9-how-does-bitcoin-mining-work` | Decentralized coordination, validation, consensus games |
| Block 5 — Transactions: Send & Receive | `6-how-to-use-bitcoin`, `7-using-bitcoin-in-daily-life` | Wallets, invoices, addresses, send/receive safety |
| Block 6 — Transaction Fees & UTXOs | `8-how-bitcoin-works/2-the-utxo-model`, `9-how-does-bitcoin-mining-work` | UTXO puzzles, coin selection, fee sizing, change outputs |
| Block 7 — Cold Storage | `6-how-to-use-bitcoin`, `10-what-future-can-bitcoin-build`, back matter | Custody, backups, threat models, recovery planning |

Each block folder has its own `CURRICULUM.md` file with detailed source links and practical game opportunities.

## Attribution rule for future EDK builds

When a game, website section, worksheet, or facilitator guide reuses My First Bitcoin curriculum wording or structure, include attribution in the relevant file/page:

> Curriculum source adapted from My First Bitcoin Bitcoin Diploma, licensed CC BY-SA 4.0.
