Facilitator rule

Let the room discover the concept.

Do not lecture first. Run the friction, ask what happened, then name the concept. That sequence is now consistent across the whole flagship system.

  • Open with failed barter.
  • Introduce a saleable good only after the room feels the pain.
  • Use Medici materials after the core lesson, not instead of it.
