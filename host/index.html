Host mode

Run Bitcoin lessons with a group.

Host mode is the facilitator side of EDK: choose a block, follow a run-of-show, project prompts, run activities, and debrief without digging through learner content.

Like Airbnb Hosting

Different mode, different job.

Learn mode is for the student. Host mode is for the person running the room.

Session formats

Pick the room you are actually in.

10 minutes

Meetup opener

One quick scenario, one group vote, one debrief. Best for introducing the idea before a talk.

30 minutes

Workshop block

Run the activity, split groups, compare answers, and connect the concept to Bitcoin.

60 minutes

Full class

Run the full sequence with roles, reflection, discussion, and extension activity.

Facilitator dashboard

Beginner funnel run-of-show

Block 1

Open with money functions

Use candy/dollar/monopoly bill prompt. Outcome: learners can explain why money works.

Block 2

Run the market plaza

Assign roles, create failed trades, introduce a saleable good, then compare money types.

Block 3

Run the inflation auction

Auction one good before and after new money enters. Debrief purchasing power.

Block 4

Match problems to Bitcoin

Give groups problem cards and solution cards. They defend the match out loud.

Materials checklist

Bring only what the activity needs.

  • Role cards or sticky notes
  • Tokens / candy / small objects for trading
  • Timer
  • Whiteboard or projected prompt
  • Optional printed scorecards
Host rule

Let the room discover the concept.

Do not lecture first. Run the friction, ask what happened, then name the concept. That is the EDK teaching style.

EDK is a static, open-source Bitcoin learning app for solo learners and group facilitators.
